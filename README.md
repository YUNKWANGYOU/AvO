# 공의 중력 가속도를 이용한 장애물 피하기 게임

### 주요기능

  1. 스크린을 반으로 나눴을 때 왼쪽을 클릭하면 공이 왼쪽으로, 오른쪽을 클릭하면 공이 오른쪽으로 튄다.
  2. 장애물을 피하면 시간에 따라 스코어가 올라간다.
  3. 장애물이 공에 닿게되면 게임이 종료되며 스코어가 기록된다.
  (자세한 구현 기능은 동영상을 참고하세요.)
